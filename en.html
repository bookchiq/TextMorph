TextMorph

Overview

TextMorph is a plugin available for mootools and jQuery which can produce forms with text. It is based on the specifications under css css 2.1 (available in English and French).

Principle

TextDraw relies on two properties present in the css standards, the float and margin-top, combining them so as to show the text along multiple div whose size is calculated for a given predetermined aspect.

Fonctionnement du plugin - de la l'aspect initial au résultat final

Two div are added together on each line. One fleet to the right and one left. An extra div is added and contains the attribute css clear: both to help write the next line. The text is still included him in a different div, hyphenation is so basic (see the explanation on the hyphenation), and set a negative margin-top size of the original div (or height parameter, if it is passed to the function).

Disponibility

This plugin is available for frameworks jquery and mootools. The demos are here for jquery and here for mootools

Hyphenations

The hyphenation is done by this plugin so simplistic, they are disabled with the parameter shy. The interest is to add a more powerful hyphenation javascript or php. Indeed, apart from not following the rules of hyphenation communnément admitted, each language has different rules which may be complex, making the function hyphenation obliterates all tags embedded in text form.

Exemple

new TextMorph(document.getElementById('circle2'),{
	shy:null
});
